void TST::tst< T >::erase ( const std::string &  key)

This method is to remove a value from the TST. You must provide the key that holds the value. If the key exist deletes the value and delete the path to it if there are no other keys that are using it. Otherwise, if the key is not there nothing will happen.

This method is used to add a new key to the TST. You must provide the key and the value to store. If the key was not before in the TST it will create the path to it and store the value. Otherwise, if the key was already in the TST it will update the value of the key with the new one.
